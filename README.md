# API Documentation
## Authentication

All API must use this authentication

Request :

- Header :
  - Authorization : "Bearer | token"

Except :

- Search Products
- List Products
- Get Products
- List Categories
- Get Categories
- Create User
- Login User

---

## Login User

<details>
  <summary>Example</summary>

Request :

- Method : POST
- Endpoint : `/api/login`
- Header :

  - Accept: application/json
  - Content-Type: application/json

- Body :

```json
{
  "email": "string",
  "password": "string"
}
```

Response :

```json
{
  "code": "number",
  "status": "string",
  "data": [
    {
      "token": "string"
    }
  ]
}
```

</details>
